Galería de Aves

Aplicación móvil desarrollada en Flutter que permite explorar una galería de aves, mostrando imágenes y posiblemente información relevante sobre cada especie.

Descripción
Galería de Aves es una app enfocada en ofrecer una experiencia visual simple e intuitiva para visualizar distintas especies de aves. Está diseñada como proyecto de aprendizaje y práctica en desarrollo móvil con Flutter.

Características:
📷 Visualización de aves en formato galería
📱 Interfaz amigable y adaptable
⚡ Rendimiento rápido gracias a Flutter
🧭 Navegación sencilla entre pantallas
